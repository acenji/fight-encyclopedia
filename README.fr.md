## Le Probleme

Des centaines d'arts martiaux possedent des techniques qui n'existent que dans **la tradition orale** — transmises de maitre a eleve sans aucun document ecrit. Si elles ne sont pas documentees, elles seront perdues.

Les ressources actuelles sont **fragmentees** : techniques de BJJ sur un site, Judo sur un autre, frappe sur un troisieme. Le meme bras de levier existe en BJJ, Judo, Sambo et MMA — mais est documente separement dans chacun, sans lien entre eux.

**Aucune ressource ne couvre toutes les techniques de combat de tous les arts dans un systeme unifie.**

Jusqu'a maintenant.

## Ce Que Nous Avons Construit

**Fight Encyclopedia** catalogue les techniques de combat de **tous les arts martiaux et sports de combat** dans une taxonomie unique, scientifiquement organisee — comme un systeme de classification biologique pour le combat.

Chaque technique est documentee avec une analyse biomecanique, des noms japonais, une evaluation du danger, la legalite en competition, des references verifiees, des contre-techniques et des chaines de preparation.
