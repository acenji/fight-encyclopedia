## 課題

数百の格闘技には、**口伝でのみ伝えられる技術**が存在します。師から弟子へと受け継がれ、文書化された記録がありません。記録されなければ、失われてしまいます。

現在のリソースは**断片化**しています：BJJの技術はあるサイトに、柔道は別のサイトに、打撃技は第三のサイトに。同じ腕十字固めがBJJ、柔道、サンボ、MMAに存在しますが、それぞれ別々に記録され、相互の関連性がありません。

**すべての格闘技の技術を統一的なシステムでカバーする単一のリソースは存在しませんでした。**

今までは。

## 私たちが構築したもの

**Fight Encyclopedia（ファイト・エンサイクロペディア）** は、**すべての格闘技と戦闘スポーツ**の技術を、科学的に組織された単一の分類体系にまとめました — 戦闘のための生物学的分類体系のようなものです。

すべての技術は、生体力学的分析、日本語名、危険度評価、競技での合法性、検証済み参考文献、カウンター技術、セットアップチェーンとともに記録されています。

## 7段階分類体系

すべてのテクニックは**生物学的分類体系**を用いて分類されています：

```
クラス ─── グループ ─── ファミリー ─── サブファミリー ─── ジーナス ─── スピーシーズ ─── バラエティ
```

**例：**

```
サブミッション > 絞め技 > 足絞め > 三角絞め
  > 三角絞め > クローズドガードからの三角 > スタンダード三角
```

柔道の*十字固め*とBJJの*アームバー*は同じ技術として分類されます。レスリングの*ダブルレッグ*と柔道の*双手刈*は同じ生体力学的ファミリーに属します。

> **[分類体系の詳細ドキュメント →](TAXONOMY.md)**

## 各テクニックの内容

各テクニックには最大**43の構造化フィールド**が9カテゴリにわたって含まれています：

<details>
<summary><b>クリックして43フィールドを展開</b></summary>

| カテゴリ | 記録内容 |
|---|---|
| **アイデンティティ** | 名称、日本語名（漢字/カタカナ）、ローマ字表記、翻訳、別名 |
| **テクニカル** | 生体力学的メカニズム、ポジションエントリー、バリアント、トレーニングノート、よくある間違い |
| **レーティング** | 危険度（1-10）、難易度、競技での合法性（IBJJF、IJF、ADCC、UFC） |
| **歴史** | 起源、有効性評価、著名な試合での使用記録 |
| **参考文献** | 一次資料、引用、URL、ページ番号 — 未検証の主張なし |
| **学術** | 出典帰属、系譜、ピアレビュー状態、最終検証日 |
| **ファイターデータ** | 必要な身体属性、カウンター技術、セットアップチェーン、競技での使用頻度 |
| **分類** | 7段階分類パス |
| **日本語** | 伝統的漢字またはカタカナ音訳、名称起源タイプ（伝統的/外来語/混合） |
