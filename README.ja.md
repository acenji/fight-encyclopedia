# Fight Encyclopedia

### 世界初の格闘技テクニック統合分類体系

[**fightencyclopedia.com**](https://fightencyclopedia.com)

[English](README.md) | **日本語** | [Portugues](README.pt.md)

---

**Fight Encyclopedia（ファイト・エンサイクロペディア）** は、**すべての格闘技と戦闘スポーツ**の技術を、科学的に組織された単一の分類体系にまとめた包括的な知識データベースです。

生物学の分類体系をベースに、格闘技のテクニックを体系化した **「格闘技のウィキペディア」** です。

<br>

## 7段階分類体系

```
クラス (Class)
 └── グループ (Group)
      └── ファミリー (Family)
           └── サブファミリー (SubFamily)
                └── ジーナス (Genus)
                     └── スピーシーズ (Species)
                          └── バラエティ (Variety)
```

<br>

## 対象格闘技（183種）

- **打撃系** (66): ボクシング、ムエタイ、空手、テコンドー、キックボクシング等
- **組技系** (42): BJJ、柔道、レスリング、サンボ、キャッチレスリング、相撲等
- **総合系** (25): MMA、クラヴマガ、ジークンドー、パンクラチオン等
- **武器系** (50): 剣道、フェンシング、HEMA、エスクリマ、居合道等
