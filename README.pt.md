# Fight Encyclopedia

### A Primeira Taxonomia Unificada de Tecnicas de Luta do Mundo

[**fightencyclopedia.com**](https://fightencyclopedia.com)

[English](README.md) | [日本語](README.ja.md) | **Portugues**

---

**Fight Encyclopedia** e um banco de dados abrangente que cataloga tecnicas de luta de **todas as artes marciais e esportes de combate** em uma unica taxonomia cientificamente organizada.

Pense como a **Wikipedia combinada com um sistema de classificacao biologica** — mas para tecnicas de luta.

## As 9 Classes de Tecnicas

| Classe | Descricao |
|---|---|
| **Finalizacao** | Estrangulamentos, chaves articulares, compressoes |
| **Golpe** | Socos, chutes, cotoveladas, joelhadas |
| **Projecao** | Projecoes de quadril, ombro, sacrificio |
| **Queda** | Single leg, double leg, rasteiras |
| **Clinch** | Agarramentos, frames, controle de clinch |
| **Posicao** | Guardas, montadas, controle lateral, controle das costas |
| **Escape e Reversao** | Escapes, raspagens, reversoes |
| **Defesa** | Bloqueios, esquivas, contra-ataques |
| **Arma** | Espada, bastao, faca, armas tradicionais |

## Artes Marciais Cobertas (183)

- **Percussao** (66): Boxe, Muay Thai, Karate, Taekwondo, Kickboxing, etc.
- **Agarramento** (42): BJJ, Judo, Wrestling, Sambo, Luta Livre, Sumo, etc.
- **Hibrido** (25): MMA, Krav Maga, JKD, Pankration, etc.
- **Armas** (50): Kendo, Esgrima, HEMA, Escrima, etc.
