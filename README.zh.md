## 问题

数百种武术拥有**仅存在于口头传统**中的技术——从师父传给弟子，没有书面记录。如果不加以记录，它们将会失传。

现有资源是**碎片化的**：BJJ技术在一个网站上，柔道在另一个，打击技术在第三个。同一个十字固在BJJ、柔道、桑搏和MMA中都存在——但在各自中被分别记录，彼此之间没有联系。

**没有任何单一资源以统一系统涵盖所有武术的所有格斗技术。**

直到现在。

## 我们构建了什么

**Fight Encyclopedia** 将**所有武术和格斗运动**的格斗技术编目到一个科学组织的统一分类体系中——就像格斗的生物分类系统。

每种技术都配有生物力学分析、日语名称、危险评级、比赛合法性、经验证的参考文献、反制技术和连接链。

每种技术使用**生物学式层级结构**进行分类——从最广泛的类别到具体的变体：

```

```
